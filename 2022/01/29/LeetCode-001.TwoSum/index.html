LeetCode - 001.Two Sum

1. 题目

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

给定一个整型数组和一个整数目标值，返回和为目标值的两个数的数组下标。

You may assume that each input would have exactly one solution, and you may not use the same element twice.

每个输入只有一个唯一的答案，并且不会用到相同的两个元素。

You can return the answer in any order.

可以以任何顺序返回答案。

2. 思路

注册 LeetCode 后的第一道题，读完题目后的第一想法就是 for循环 不就完事了。后来写完一翻评论区，果然小白就是小白啊~哈哈哈。

2.1 枚举法

枚举法应该是最容易想到的一个思路，也是时间复杂度最高的一个方法：

  • 通过双层 for循环来遍历出数组中的每一种组合方式并将他们求和。
  • 与目标值比较得出结果返回。
  • 时间复杂度：O(n^2^)
  • 空间复杂度：O(1)

2.2 哈希表法

这个方法是评论区看到的：

  • 新建一个哈希表，将数组中的第一个数和下标添加进哈希表中。
  • 添加第二个数x 时，将 target - x 与哈希表中已存在的数查找。
    • 如果存在，则说明 x 与 查找到的数 即为答案，返回这两个数的下标即可。
    • 若不存在，则将这个数 x 与其下标添加进哈希表中，继续判断下一个数。
  • 时间复杂度：O(n)
  • 空间复杂度：O(n)
