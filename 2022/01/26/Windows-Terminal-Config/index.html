Windows Terminal 配置及美化记录

电脑经常更换重装，记录一下配置过程以供参考。

1. PowerShell 配置与美化

我的大部分操作都在 pwsh 上完成，不使用的可以跳过这一部分，自行寻找相关 shell 的美化配置教程。

1.1 安装 PowerShell 7.x

1.1.1 安装方法一：msi 安装包

推荐使用 LTS 版本，官方下载地址 GitHub

下载相应版本安装即可

1.1.2 安装方法二：dotnet 安装

如果电脑中有 .NET Core SDK 或者 Visual Studio 20XX with .NET，则可以作为 .NET全局工具 来安装

在 Shell 中执行安装命令：dotnet tool install --global PowerShell

1.2 配置 oh-my-posh

1.2.1 安装

  • 使用 PowerShell 一键安装：Install-Module oh-my-posh

1.2.2 配置

  • 执行命令，编辑配置文件

  • 文件中添加

    如果不是很喜欢这个主题，可以使用以下命令来查看所有主题：Get-PoshThemes
    找到喜欢的主题后，可以在第二行中将主题名称替换为你想要主题的名称。

  • 保存并退出，重启 PowerShell，效果如下

1.2.3 安装Nerd Fonts

可以看出上图有不少乱码，因为默认字体不支持部分图标的显示，我们需要安装合适的字体给 PowerShell 使用

官网地址

1.3 配置 PowerShell 字体

  • 在 Windows Terminal 中使用 Ctrl + , 打开设置

  • 选择PowerShell - 外观 - 字体 - 选择刚刚安装的字体

  • 保存并重启 Windows Terminal

1.4 VS Code 正确配置 PowerShell

  1. 打开 VS Code 的 设置

  2. 搜索 Integrated:Font

  3. 将 NF 字体名称填入 Integrated:Font Family 中

  4. 保存并重启 VS Code，在终端中选择 pwsh 即可
